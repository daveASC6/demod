About The App - Streem Performers are a huge part of the New Yorker culture. They could be either found all over New York, but the problem is we never know when or where they will be, we just happen to be there to see their shows. This web app solves that problem by alerting users where the street performers are in a mile radius from them using geolocation. When this happens the user would be able to get to the performance in time to see it.
